# Portfolio Pro 版本日志

## 版本控制规范

### 版本号规则：主版本.次版本.修订号
- **主版本**：重大功能重构或架构升级
- **次版本**：新增功能模块
- **修订号**：Bug修复、性能优化

---

## v5.3.0 (2026-03-11)

### ✨ 新增功能
- **实时行情抓取**: 对接新浪财经 API，一键获取股票实时价格
- **飞书消息推送**: 支持每日持仓简报卡片推送到飞书群
- **同步面板**: 点击「同步」按钮可选择多种操作

### 📁 新增文件
- `data_fetcher.js` - 行情数据抓取模块
- `feishu_notifier.js` - 飞书消息推送模块

### 🔧 使用方法
1. **抓取实时行情**: 点击「同步」→ 选择「1」→ 自动获取持仓股票最新价格
2. **配置飞书**: 点击「同步」→ 选择「3」→ 粘贴 Webhook 地址
3. **发送测试**: 点击「同步」→ 选择「2」→ 发送持仓简报到飞书
